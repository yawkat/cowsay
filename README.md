Cowsay
======

Cowsay plugin for bukkit and bungee.

Usage
-----

`/cowsay <message>`

`/cowthink <message>`

Screenshots
-----------

![Console](http://s.yawk.at/v4s9.png)

![In-Game](http://s.yawk.at/Pc9n.png)

Permissions
-----------

- `cowsay.cowsay`: Give access to `/cowsay`
- `cowsay.cowthink`: Give access to `/cowthink`
